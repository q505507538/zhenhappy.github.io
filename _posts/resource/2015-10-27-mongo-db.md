## 基本数据库操作

### 数据库列表

{% highlight js %}
show dbs
{% endhighlight %}

### 查看当前连接的数据库

{% highlight js %}
db
{% endhighlight %}

### 切换数据库

{% highlight js %}
use test
{% endhighlight %}

### 查看数据表

{% highlight js %}
show collections
{% endhighlight %}
